JS中的随机数

发表于 2016-05-22 | 分类于 Javascript

JS产生随机数的方法通常采用的是Javascript的Math.random()函数。
常用的几种用法：
Math.random()表示结果为0-1间的一个随机数(包括0,不包括1)
Math.floor(Math.random()*10+1)表示结果为1-10间的一个随机数
Math.floor(Math.random()*24)表示结果为0-23间的一个随机数

Math.random()

返回0和1之间的伪随机数，可能为0，但总是小于1，即：[0,1)

返回指定范围的随机数(m~n之间)的公式

Math.ceil()

向上取整，获取1~10的随机数

Math.round(n)
返回n四舍五入后整数的值
